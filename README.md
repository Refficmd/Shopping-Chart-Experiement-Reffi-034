# 🛍 2. Cart Page

## Halaman ini menampilkan daftar produk yang telah ditambahkan ke dalam keranjang.

- ## Setiap item menampilkan:
  ### - Nama produk
  ### - Harga satuan
  ### - Jumlah (quantity)
  ### - Subtotal per item

- ## Pengguna dapat:
  ### - Menambah jumlah produk.
  ### - Mengurangi jumlah produk.
  ### - Menghapus seluruh isi keranjang menggunakan tombol clear.
  ### - Total harga seluruh produk ditampilkan pada bagian bawah halaman.

- ## Jika keranjang kosong, aplikasi akan menampilkan:
  ### - Tampilan UI kosong.
  ### -  Tombol “Lanjut Belanja” untuk kembali ke halaman produk.
