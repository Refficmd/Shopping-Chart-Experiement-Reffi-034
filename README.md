# Shopping-Chart-Experiement-Reffi-034

1. Product List Page
   - User Bisa:
     - Mencari produk Berdasarkan nama
     - Filter produk berdasarkan kategori
     - Menambahkan Produk ke Keranjang
    - Angka di icon keranjang menunjukkan *total* dari seluruh produk yang ada di keranjang   

2. Cart Page
   - Tampilan
     - Nama Produk
     - Harga
     - Jumlah
     - Subtotal per item
   - User bisa:
     - Tambah jumlah produk
     - Mengurangi Jumah Produk
     - Menghapus seluruh produk yang ada di keranjang
   - Total harga ditunjukkan di bawah
   - Jika keranjang kosong:
     - Menunjukkan ui kosong
     - Tombol untuk lanjut belanja
       
3. Checkout Page
   - Menunjukkan total order
   - Menunjukkan total harga
   - Form pengiriman
     - Nama Lengkap
     - Nomor Telepon
     - Alamat
   - Jika berhasil
     - Dialog succes akan muncul
     - Keranjang akan kosong
     - Kembali ke laman product
